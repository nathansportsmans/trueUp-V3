FAQ's Help
Where do I Begin? Make sure to start by following the instructions up above these FAQ's. These instructions will give the best insigt on how to use this software.
How do I make my Spreadsheet into a .csv format? While in excel, navigate to the 'file' tab on the top left of the screen. Then click 'export.' Click on the option to 'Change File Type' and you will be able to choose '.csv'. After you have this selected save the file.
Can I Drag and Drop files? Yes you can. Just simply click the upload buttons, or use the drag and drop feature by dragging files from your computer and dropping them in the button fields.
Do I need to delete all the columns with commas in them for this to work? Yes you do. If you don't you will get some weird issues.
Why is GSUITE not working? GSUITE is one of the more particular True-Up Formats. In the CSV export from GSUITE, it titles the columns with "[REQUIRED]". For example: "First Name [REQUIRED]." You absolutely have to change this formating by simply erasing the "[REQUIRED]" part.
Where do I get the CSV for AWS users? Easy! Just ask Nathan for the list. You cannot export the user list using Amazon's current software, but Nathan can use javascript to pull the information from the UI online.
I clicked submit. Why did nothing happen? You have to double click it for the trueup to work.
Where do I get the Employee list that includes contractors? Easy! Just ask Nathan for the list.
